> [!IMPORTANT]
> **Degree Centrality Formula (must memorize!):**
> - Undirected: `CD(v) = deg(v) / (n - 1)`
> - Directed: Calculate In-degree and Out-degree separately
>
> **Example:** In a network of 5 nodes, if node A is connected to 3 others:
> `CD(A) = 3 / (5-1) = 3/4 = 0.75`

> [!TIP]
> **Granovetter's Key Insight:** "Weak ties are more valuable for spreading NEW information because they bridge different social groups. Strong ties share redundant information within the same cluster."
